# Chat Overlay

A browser extension overlay for Telegram Web that shows AI-generated reply suggestions and a Stockfish-style conversation score bar.

## What it does

- Injects a sidebar panel into Telegram Web
- Scores the current conversation in real time
- Surfaces the top reply suggestions ranked by quality
