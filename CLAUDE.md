## Project Overview

**Chatfish** is a Go application that reads conversations (Telegram integration), generates high-quality reply suggestions, and scores them — like "Stockfish for chat".

## Technology Stack

- **Language**: Go
- **Integration**: Telegram (conversation source)
- **Purpose**: AI-powered reply suggestion and scoring
